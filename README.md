# Greenland Customizations (Odoo 17 ERP)

A production-focused deployment specializing in targeted functional adjustments and relational database quality assurance for a live client's Odoo 17 ERP ecosystem.

## 🛠️ Project Scope & Contributions

### 📊 Database Customization & Configurations
- **Feature Adjustments:** Performed direct database-level feature editing within core Odoo applications to meet specific client requirements.
- **Module Tuning:** Successfully customized and adjusted key transactional fields, form configurations, and workflows within the **Purchase** and **Sales** modules to maximize operational workflow efficiency.

### 🧪 Quality Assurance & Database Testing
- **Schema Validation:** Executed targeted functional testing to ensure seamless relational data integrity between purchase orders, sales pipelines, and database records.
- **Workflow Testing:** Systematically tested default date handling, automated expense validation, and manual data-entry configurations to verify stability and prevent workflow blocks before client handover.

## 🧰 Tech Stack
- **ERP Platform:** Odoo 17 Community / Enterprise
- **Backend & Database:** Python | PostgreSQL
- **Dev Tools:** Git | Relational Database Testing
